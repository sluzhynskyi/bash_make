## Tests

```bash
./shell_make.sh -f test/shmakefile1
./shell_make.sh -f test/shmakefile0    
```
This would make the same to ```make f test/shmakefile1``` or  ```make f test/shmakefile0```

## Usage

```bash
./shell_make.sh [-h|--help] [-f|--file <file>] <target1> <target2> ... <targetN>
```
